# 🌤️ Weather Dashboard

A dynamic weather application that provides real-time weather updates for different cities using a Weather API.

## ✨ Features
* **Real-time Data:** Fetches current weather conditions and forecasts.
* **Search Functionality:** Look up the weather for any city worldwide.
* **Dynamic Icons:** Displays weather icons (sunny, rainy, cloudy) based on the current state.
* **Responsive Layout:** Optimized for all devices and screen sizes.

## 🛠️ Tech Stack
* **HTML5 & CSS3**
* **JavaScript** (ES6+)
* **Weather API** (e.g., OpenWeatherMap)
* **Bootstrap** (Optional - *لو كنتِ استخدمتيه*)
